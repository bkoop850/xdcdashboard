# Filament React SPA with Vite

## Available scripts

| Command                |                     |                     | Description                                        |
| ---------------------- | ------------------- | ------------------- | -------------------------------------------------- |
| `npm run dev`          | `yarn dev`          | `pnpm dev`          | Runs the app in development mode                   |
| `npm run build`        | `yarn build`        | `pnpm build`        | Builds the app for production to the `dist` folder |
| `npm run preview`      | `yarn preview`      | `pnpm preview`      | Runs the production build locally                  |
| `npm test`             | `yarn test`         | `pnpm test`         | Launches the test runner                           |
| `npm run lint`         | `yarn lint`         | `pnpm lint`         | Runs ESLint to check code quality                  |
| `npm run format`       | `yarn format`       | `pnpm format`       | Formats code with Prettier                         |
| `npm run format-check` | `yarn format-check` | `pnpm format-check` | Checks if code is formatted correctly              |
| `npm run type-check`   | `yarn type-check`   | `pnpm type-check`   | Runs TypeScript type checking                      |
